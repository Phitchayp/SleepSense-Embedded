# Sleep Sense
เครื่องวัดคุณภาพการนอนด้วยความเข้มของแสงและความดังของเสียง

โปรแกรมที่ใช้
* Arduino
* Arduino Cloud

อุปกรณ์ที่ใช้
* ESP32
* Sensorตรวจจับเสียง INMP441
* Sensorตรวจจับแสง BH1750FVI

# Hardware
* Schematic
* PCB
* ต่ออุปกรณ์ Sensor INMP441 , Sensor BH1750FVI
